# ASP.NET Task

A full-stack web application built with ASP.NET MVC and .NET Core Web API.

## Features
- User registration and login
- JWT token authentication
- BCrypt password hashing
- View and edit profile
- SQL Server database with Entity Framework Core

## Projects
- `UserApi` — .NET Core Web API (backend)
- `UserMvcApp` — ASP.NET MVC (frontend)

## How to Run
1. Open solution in Visual Studio
2. Run `Update-Database` in Package Manager Console
3. Set both projects to start simultaneously
4. Press F5
